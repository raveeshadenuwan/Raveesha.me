GitHub Pages Portfolio Project

Files included:
- index.html
- style.css
- script.js
- cv.pdf
- yourphoto.jpg

Before publishing:
1. Replace YOUR_FORM_ID in index.html with your Formspree form id.
   Example:
   https://formspree.io/f/xqabcdyy

2. Replace:
   - https://github.com/yourusername
   - https://linkedin.com/in/yourprofile

3. Replace yourphoto.jpg with your real photo.
4. Replace cv.pdf with your real CV.

How to publish on GitHub Pages:
1. Create a new GitHub repository.
2. Upload all files.
3. Go to Settings > Pages.
4. Select branch: main
5. Save.

Your site will then be live.
